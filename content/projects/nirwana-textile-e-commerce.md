Gateway Nirwana Textile is a web platform created to serve as a digital gateway for the textile business operations, enabling users (e.g. internal staff, partners, or clients) to access information about products, orders, material sourcing, inventory status, or related services (depending on the features implemented).

This gateway acts as a central interface, bridging backend systems and users, ensuring smooth data flow and usability.

#### My Responsibilites & Contributions

1. Leading a team of frontend dev to deliver production-ready web application.
2. Ensuring the site adapts seamlessly across desktop, tablet, and mobile screens.
3. Communicating with backend endpoints to fetch data and render dynamic content.
4. Managing application state via Redux to handle user sessions, data fetching, and caching.
5. Lazy-loading, code-splitting, optimizing assets to reduce load time and improve user experience.
6. Working with backend devs, designers, and stakeholders to refine UI behaviors, handle edge cases, and deliver a polished product.
